# COSC_301_Group_4_Instacart_Basket_Analysis
Project for COSC 301 2025 Winter Session Term 2.

[Kaggle Instacart Dataset Link] (https://www.kaggle.com/datasets/psparks/instacart-market-basket-analysis?resource=download)

Project Name: Instacart Market Basket Analysis

Steps to reproduce the project:

1. Run the following in your terminal to clone the repository to your local computer.
   `git clone https://github.com/aadilshaji/COSC_301_Group_4_Instacart_Basket_Analysis.git`

2. cd into the repository in your terminal
   `cd COSC_301_Group_4_Instacart_Basket_Analysis`

3. Run the script file in your terminal:
   `bash scripts/get_data.sh`

4. Run the main.ipynb file in Visual Studio Code using the "Run all" button at the top of the file when you open it in VS code.

5. Results for each code cell will be seen below each cell.
